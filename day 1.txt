Q1.Explore and Explain various methods in console function.
A1. 1) console.log():used to print the output to the console and can be used for testing.
      Eg:- console.log('abc'); 
	console.log(1); 
	console.log(true); 
	console.log(null); 
	console.log(undefined); 
    2)console.warn():writes a warning to the console.The warning message will be highlighted with yellow color.
      Eg:- console.warn("THIS IS WARNING");
    3)console.error():writes an error message to the console. Useful in testing of code. 
      Eg:-console.error('This is a simple error');		
    4)console.clear():clears the console and also write a message in the console: "Console was cleared". 
      Eg:-console.clear();
    5)console.table():allows  to generate a table inside a console. The input must be an array or an object which will be shown as a table.
      Eg:-console.table({'a':1, 'b':2}); 		


Q2.Write the difference between var,let and const with code examples.
A2. 1)var declarations are  function scoped
    let and const are block scoped.
    2)var variables can be updated and re-declared within its scope,
      let variables can be updated but not re-declared,
      const variables can neither be updated nor re-declared.
    3) var and let can be declared without being initialized, 
       const must be initialized during declaration.
