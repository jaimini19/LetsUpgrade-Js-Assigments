Q2.Explain with examples the remaining methods of string and array.
A2.search():- method searches a string for a specified value and returns the position of the match.
   Eg:-let str = "Hello we are Learning JS!";
       let pos = str.search("JS");
       console.log(pos);
   Output:23
   slice():- extracts a part of a string and returns the extracted part in a new string.
   Eg:-let str = "Volvo,BMW, Kiwi";
       let res = str.slice(7, 10);
       console.log(res);
   Output:-BMW
   substring():-similar to slice but cannot accept negative indexes.
   Eg:-let str = "Volvo,BMW, Kiwi";
       let res = str.slice(7);
       console.log(res); 
   Output:-BMW,Kiwi
   replace():-method replaces a specified value with another value in a string:
   Eg:-let str = Lets learn Java!";
       let n = str.replace("Java", "JS");
       console.log(n);
   Output:-Lets learn JS.
